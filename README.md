# Portafolio Personal en React (María Girón)

Este es un proyecto de portafolio personal desarrollado con **React** y **Vite**, como parte de la actividad académica solicitada. Aquí se presentan mis datos personales, estudios, habilidades y medios de contacto, todo organizado en componentes funcionales.

## 🚀 Tecnologías Utilizadas
- React
- Vite
- CSS personalizado
- Bootstrap (para diseño responsive)

## 🧱 Estructura del Proyecto

```plaintext
portafolio-react/
│

│

📦 Instalación y Ejecución del Proyecto
Sigue estos pasos para ejecutar el portafolio localmente:

# Clona el repositorio (si ya lo subiste a GitHub)
git clone https://github.com/tu-usuario/portafolio-react.git
cd portafolio-react

# Instala las dependencias
npm install

# Ejecuta el proyecto en modo desarrollo
npm run dev
